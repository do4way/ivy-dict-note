Ivy dictionary note
---
实现查阅webster learning的网页中的英语单词， 并将单词释义和发音记录到Evernote笔记中。

用法
```
$ go run note.go -h
  -n string
    	The note title
  -nb string
    	The note book name
  -note string
    	The note title
  -note_book string
    	The note book name
  -t value
    	The note tag
  -token string
    	The evernote dev token
```
